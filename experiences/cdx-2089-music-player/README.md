A skin-adaptive 3D music player with five futuristic disc decks, audio-reactive visuals, and an unfolding dance floor.

[Live demo](https://play.mint.gg/cdx-2089-music-player) · [Source code](https://github.com/mintdotgg/mint-playground/tree/main/experiences/cdx-2089-music-player)

Made with [Mint MCP](https://mcp.mint.gg/) as the 3D asset pipeline and
[Mint Three.js Skills](https://github.com/mintdotgg/mint-threejs-skills) as the coding workflow.

## Run locally

From the `mint-playground` repository root:

```bash
pnpm install
pnpm --dir experiences/cdx-2089-music-player dev
```
