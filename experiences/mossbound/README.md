A first-person fantasy melee run through five escalating waves of corrupted grove creatures, ending in a showdown with the Thornheart.

[Live demo](https://play.mint.gg/mossbound) · [Source code](https://github.com/mintdotgg/mint-playground/tree/main/experiences/mossbound)

Made with [Mint MCP](https://mcp.mint.gg/) as the 3D asset pipeline and
[Mint Three.js Skills](https://github.com/mintdotgg/mint-threejs-skills) as the coding workflow.

## Run locally

From the `mint-playground` repository root:

```bash
pnpm install
pnpm --dir experiences/mossbound dev
```
