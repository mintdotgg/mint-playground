A source-aware 3D atlas for exploring all 64 official San Francisco public tennis facilities.

[Live demo](https://play.mint.gg/sf-tennis-atlas) · [Source code](https://github.com/mintdotgg/mint-playground/tree/main/experiences/sf-tennis-atlas)

Made with [Mint MCP](https://mcp.mint.gg/) as the 3D asset pipeline and
[Mint Three.js Skills](https://github.com/mintdotgg/mint-threejs-skills) as the coding workflow.

## Run locally

From the `mint-playground` repository root:

```bash
pnpm install
pnpm --dir experiences/sf-tennis-atlas dev
```
