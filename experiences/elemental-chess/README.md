# Elemental Chess

Choose Water, Earth, Fire, or Air, face a rival, and play a complete cinematic 3D chess match in a Mint-authored elemental world.

[Live demo](https://play.mint.gg/elemental-chess) · [Source code](https://github.com/mintdotgg/mint-playground/tree/main/experiences/elemental-chess)

Made with [Mint MCP](https://mcp.mint.gg/) as the 3D asset pipeline and
[Mint Three.js Skills](https://github.com/mintdotgg/mint-threejs-skills) as the coding workflow.

## Run locally

From the `mint-playground` repository root:

```bash
pnpm install
pnpm --dir experiences/elemental-chess dev
```
