An immersive collector vault for selecting and inspecting six fictional premium sports trading cards.

[Live demo](https://play.mint.gg/dynasty-index) · [Source code](https://github.com/mintdotgg/mint-playground/tree/main/experiences/dynasty-index)

Made with [Mint MCP](https://mcp.mint.gg/) as the 3D asset pipeline and
[Mint Three.js Skills](https://github.com/mintdotgg/mint-threejs-skills) as the coding workflow.

## Run locally

From the `mint-playground` repository root:

```bash
pnpm install
pnpm --dir experiences/dynasty-index dev
```
