# Mint Hoops

A 60-second arcade basketball score attack with drag-to-shoot controls, combo multipliers, and a three-point finale.

[Live demo](https://play.mint.gg/mint-hoops) · [Source code](https://github.com/mintdotgg/mint-playground/tree/main/experiences/mint-hoops)

Made with [Mint MCP](https://mcp.mint.gg/) as the 3D asset pipeline and
[Mint Three.js Skills](https://github.com/mintdotgg/mint-threejs-skills) as the coding workflow.

## Run locally

From the `mint-playground` repository root:

```bash
pnpm install
pnpm --dir experiences/mint-hoops dev
```
