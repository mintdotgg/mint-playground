# Pogo Man

<a href="https://play.mint.gg/pogo-man">
  <img src="https://play.mint.gg/experience-assets/pogo-man/social-card.webp" alt="A stylized rider bounces on a pogo stick over yellow and coral cars on a colorful city street." width="100%">
</a>

Charge double jumps to clear city traffic, build combos, and chase a high score in a one-button 3D arcade run.

[Live demo](https://play.mint.gg/pogo-man) · [Source code](https://github.com/mintdotgg/mint-playground/tree/main/experiences/pogo-man)

Made with [Mint MCP](https://mcp.mint.gg/) as the 3D asset pipeline and
[Mint Three.js Skills](https://github.com/mintdotgg/mint-threejs-skills) as the coding workflow.

## Run locally

From the `mint-playground` repository root:

```bash
pnpm install
pnpm --dir experiences/pogo-man dev
```
