A cinematic travel magazine spanning five explorable Mint-generated Gaussian-splat worlds.

[Live demo](https://play.mint.gg/impossible-places) · [Source code](https://github.com/mintdotgg/mint-playground/tree/main/experiences/impossible-places)

Made with [Mint MCP](https://mcp.mint.gg/) as the 3D asset pipeline and
[Mint Three.js Skills](https://github.com/mintdotgg/mint-threejs-skills) as the coding workflow.

## Run locally

From the `mint-playground` repository root:

```bash
pnpm install
pnpm --dir experiences/impossible-places dev
```
