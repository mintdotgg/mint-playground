# Sole Archive

<a href="https://play.mint.gg/sole-archive">

Inspect and configure six fictional sneakers in an underground boutique with exploded materials, comparison, wishlist, sizing, and a persistent demo cart.

[Live demo](https://play.mint.gg/sole-archive) · [Source code](https://github.com/mintdotgg/mint-playground/tree/main/experiences/sole-archive)

Made with [Mint MCP](https://mcp.mint.gg/) as the 3D asset pipeline and
[Mint Three.js Skills](https://github.com/mintdotgg/mint-threejs-skills) as the coding workflow.

## Run locally

From the `mint-playground` repository root:

```bash
pnpm install
pnpm --dir experiences/sole-archive dev
```
