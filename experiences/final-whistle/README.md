# Final Whistle: Spain vs Argentina

Lead Spain through a 150-second 3D arcade football final against Argentina with passing, tackles, charged shots, and golden-goal drama.

[Live demo](https://play.mint.gg/final-whistle) · [Source code](https://github.com/mintdotgg/mint-playground/tree/main/experiences/final-whistle)

Made with [Mint MCP](https://mcp.mint.gg/) as the 3D asset pipeline and
[Mint Three.js Skills](https://github.com/mintdotgg/mint-threejs-skills) as the coding workflow.

## Run locally

From the `mint-playground` repository root:

```bash
pnpm install
pnpm --dir experiences/final-whistle dev
```
