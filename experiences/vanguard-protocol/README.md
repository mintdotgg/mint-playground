A cinematic five-character combat roster with animated fighters and reactive tactical staging.

[Live demo](https://play.mint.gg/vanguard-protocol) · [Source code](https://github.com/mintdotgg/mint-playground/tree/main/experiences/vanguard-protocol)

Made with [Mint MCP](https://mcp.mint.gg/) as the 3D asset pipeline and
[Mint Three.js Skills](https://github.com/mintdotgg/mint-threejs-skills) as the coding workflow.

## Run locally

From the `mint-playground` repository root:

```bash
pnpm install
pnpm --dir experiences/vanguard-protocol dev
```
