An interactive poster gallery where four sci-fi relics reshape the typography, lighting, materials, and sound.

[Live demo](https://play.mint.gg/neon-relic) · [Source code](https://github.com/mintdotgg/mint-playground/tree/main/experiences/neon-relic)

Made with [Mint MCP](https://mcp.mint.gg/) as the 3D asset pipeline and
[Mint Three.js Skills](https://github.com/mintdotgg/mint-threejs-skills) as the coding workflow.

## Run locally

From the `mint-playground` repository root:

```bash
pnpm install
pnpm --dir experiences/neon-relic dev
```
