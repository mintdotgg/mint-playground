A luminous orbital-vehicle showroom for inspecting six futuristic craft and their onboard systems.

[Live demo](https://play.mint.gg/helios-form) · [Source code](https://github.com/mintdotgg/mint-playground/tree/main/experiences/helios-form)

Made with [Mint MCP](https://mcp.mint.gg/) as the 3D asset pipeline and
[Mint Three.js Skills](https://github.com/mintdotgg/mint-threejs-skills) as the coding workflow.

## Run locally

From the `mint-playground` repository root:

```bash
pnpm install
pnpm --dir experiences/helios-form dev
```
