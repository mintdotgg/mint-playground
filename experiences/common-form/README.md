# Common Form

<a href="https://play.mint.gg/common-form">

Explore six considered everyday objects in a warm interactive showroom, inspect them in 3D, and build a persistent demo cart.

[Live demo](https://play.mint.gg/common-form) · [Source code](https://github.com/mintdotgg/mint-playground/tree/main/experiences/common-form)

Made with [Mint MCP](https://mcp.mint.gg/) as the 3D asset pipeline and
[Mint Three.js Skills](https://github.com/mintdotgg/mint-threejs-skills) as the coding workflow.

## Run locally

From the `mint-playground` repository root:

```bash
pnpm install
pnpm --dir experiences/common-form dev
```
