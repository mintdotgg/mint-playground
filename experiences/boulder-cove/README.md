# Boulder Cove

A cozy prehistoric island life game where you gather, fish, catch bugs, plant trees, and help dinosaur neighbors earn your own boulder hut.

[Live demo](https://play.mint.gg/boulder-cove) · [Source code](https://github.com/mintdotgg/mint-playground/tree/main/experiences/boulder-cove)

Made with [Mint MCP](https://mcp.mint.gg/) as the 3D asset pipeline and
[Mint Three.js Skills](https://github.com/mintdotgg/mint-threejs-skills) as the coding workflow.

## Run locally

From the `mint-playground` repository root:

```bash
pnpm install
pnpm --dir experiences/boulder-cove dev
```
